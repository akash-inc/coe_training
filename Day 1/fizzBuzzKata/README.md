# FizzBuzz Kata (TypeScript)

A small TDD-style FizzBuzz kata built with TypeScript and Jest.

## Tech Stack

- TypeScript
- Jest
- ts-jest

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run tests:

```bash
npm test
```

## Current Rules

The `fizzBuzz(n)` function returns:

- `"FizzBuzz"` when `n` is divisible by both 3 and 5
- `"Fizz"` when `n` is divisible by 3
- `"Buzz"` when `n` is divisible by 5
- `n` otherwise

## Project Structure

- `src/fizzBuzz.ts` - FizzBuzz implementation
- `src/fizzBuzz.test.ts` - Jest test cases
